# 心理学RAGアシスタント - 導入ガイド

このフォルダには、心理学の主要4学派（アドラー、フロイト、ユング、認知学派）の理論に基づき回答するAIチャットボットの全データが含まれています。
ZIPファイルを解凍したあと、以下の手順に従うだけで、あなたのPCですぐに実行できます。

---

## 🚀 準備と起動の手順（Windows向け）

専門的な知識は不要です。以下の3つのステップを順番に進めてください。

### STEP 1. Pythonのインストール
PCにPython（パイソン）が入っていない場合は、インストールしてください。
1. [Python公式サイト](https://www.python.org/downloads/) から最新版をダウンロードして実行します。
2. **重要**: インストール画面の最初に出てくる **「Add Python to PATH」** に必ずチェックを入れてから「Install Now」を押してください。

### STEP 2. 必要な部品のインストール
フォルダを開いた状態で、アドレスバーに `cmd` と入力してEnterキーを押すか、右クリックで「ターミナルで開く」を選択し、以下の1行をコピー＆ペーストしてEnterを押してください。

```powershell
pip install -r requirements.txt
```

### STEP 3. APIキーの設定
AI（OpenAI）を利用するための合鍵を設定します。
1. フォルダ内にある `.env.example` というファイルをコピーして、同じ場所に貼り付けます。
2. できたコピーのファイル名を **`.env`** に書き換えます（`.example` を消します）。
3. `.env` ファイルをメモ帳などで開き、`OPENAI_API_KEY=` の右側にあなたのAPIキーを貼り付けて保存します。

---

## 🏃 アプリの起動方法

準備ができたら、ふたたびターミナル（黒い画面）で以下のコマンドを入力します。

```powershell
python -m uvicorn app.main:app
```

画面に `Application startup complete` と表示されたら成功です！
ブラウザ（ChromeやEdgeなど）を立ち上げ、以下のURLを入力してください。

👉 **[http://127.0.0.1:8000/](http://127.0.0.1:8000/)**

---

## 💡 使いかたのヒント

- **ケース検討モード**: 4つの学派すべてから、あなたの相談に対する見解を同時に得られます。
- **用語解説モード**: 特定の用語について、最も詳しい1つの学派が深掘りして解説します。
- **終了する方法**: ターミナルの画面で `Ctrlキー` を押しながら `C` を押すと終了します。

---

## ⚠️ 困ったときは
- **「pythonが見つかりません」と出る**: STEP 1の「Add Python to PATH」にチェックを入れ忘れている可能性があります。一度アンインストールして入れ直すのが確実です。
- **APIキーのエラーが出る**: `.env` ファイルの名前が正しく `.env` になっているか、キーの前後に余計なスペースが入っていないか確認してください。
