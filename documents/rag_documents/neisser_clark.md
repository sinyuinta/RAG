## 1. 理論の骨格（本体）

### 1.1 基本前提

アンディ・クラーク（1957-）は、認知科学の哲学者であり、「拡張された心（Extended Mind）」と「予測処理（Predictive Processing）」の理論で知られる。認知を脳の内部に閉じ込める従来の見方を根本的に問い直し、認知科学のパラダイム転換を推進している。

**（1）認知は脳を超えて拡張する**

認知は頭蓋骨の中に閉じ込められていない。道具、技術、環境、他者との相互作用を含む拡張されたシステムとして認知を理解すべきである。

**（2）脳は予測機械である**

脳の基本的な機能は、感覚入力を受動的に処理することではなく、世界についての予測を生成し、予測誤差を最小化することである。

**（3）身体化された認知**

認知は抽象的な記号操作ではなく、身体と環境との相互作用に根ざしている。

**（4）能動的推論**

知覚、行動、学習はすべて、予測誤差を最小化するという単一の原理で統一的に理解できる。

### 1.2 中核概念

**（1）拡張された心（Extended Mind）**

1998年にデイヴィッド・チャーマーズとの共著論文で提唱された概念。

**基本的議論**

「オットーとインガ」の思考実験：
- インガは美術館の場所を記憶から思い出す
- アルツハイマー病のオットーはノートを見て確認する
- 両者の認知的機能は同等ではないか？

もし内的な記憶が信念を構成するなら、オットーのノートも（適切な条件下で）信念を構成すると言うべきではないか。

**パリティ原則（Parity Principle）**

「もしある過程が頭の中で起こったとしたら認知過程と見なされるような仕方で、世界の一部が機能しているなら、その世界の一部は認知過程の一部である」

**拡張の条件**

すべての外的ツールが認知の一部になるわけではない。条件：
- 信頼可能性：必要なときにアクセスできる
- 自動的なアクセス：意識的な努力なしに利用できる
- 情報への信頼：その情報を信頼している
- 過去の承認：その情報を以前受け入れた

**含意**

- スマートフォンは私たちの認知システムの一部かもしれない
- 認知障害の定義が変わりうる
- 認知的エンハンスメントの倫理的問題

**（2）予測処理（Predictive Processing）/ 予測符号化（Predictive Coding）**

クラークが近年最も力を入れている理論的枠組み。ヘルムホルツの「無意識的推論」、ベイズ推論、自由エネルギー原理（カール・フリストン）を統合。

**基本的アイデア**

脳は世界の階層的な生成モデルを持ち、そのモデルに基づいて感覚入力を予測する。実際の感覚入力と予測の差（予測誤差）が、モデルを更新するための信号となる。

**階層的予測**

```
高次領域 → 予測 → 低次領域
         ← 予測誤差 ←

高次領域は低次領域への予測を生成
低次領域は予測誤差を高次領域に送る
この過程が階層全体で繰り返される
```

**知覚は「制御された幻覚」**

知覚は外界の直接的な反映ではなく、脳が生成した「最良の推測」である。感覚入力は、この推測を制約・修正する役割を果たす。

**予測誤差の最小化の二つの方法**

1. **知覚的推論**：モデル（予測）を更新して、感覚入力に合わせる
2. **能動的推論**：行動によって世界を変え、予測に合わせる

これにより、知覚と行動が統一的に理解できる。

**精度重み付け（Precision Weighting）**

すべての予測誤差が等しく扱われるわけではない。信頼できる（精度が高い）誤差は重視され、ノイズが多い（精度が低い）誤差は無視される。

注意は、特定の誤差信号の精度を高める操作として理解できる。

**（3）能動的推論（Active Inference）**

カール・フリストンの自由エネルギー原理に基づく行動の理論。

**基本的アイデア**

生物は自由エネルギー（≒予測誤差）を最小化するように行動する。

行動は：
- 予期される感覚状態を実現するための動き
- 予測を自己成就させるプロセス
- 世界を予測に合わせる試み

例：空腹という予測誤差 → 食べ物を探す行動 → 満腹という予測状態の実現

**（4）4E認知（4E Cognition）**

クラークは、認知科学における「4E」運動の主要な推進者の一人。

**4つのE**

1. **Embodied（身体化された）**：認知は身体に根ざしている
2. **Embedded（埋め込まれた）**：認知は環境に埋め込まれている
3. **Enacted（行為遂行的）**：認知は行為を通じて遂行される
4. **Extended（拡張された）**：認知は脳を超えて拡張される

これらは相互に関連し、従来の「脳中心主義」に対抗する。

**（5）ソフトなアセンブリ（Soft Assembly）**

認知システムは、固定的な構造ではなく、状況に応じて柔軟に組み立てられる。

例：
- 同じ課題でも、利用可能なツールによって異なる認知戦略を使う
- 環境のリソースを動的に活用する
- 脳、身体、環境の境界は流動的

```
拡張された心

├── 道具、技術、環境が認知の一部になりうる
└── 認知の「肌と頭蓋骨の境界」への懐疑

↓

予測処理
├── 脳は予測機械
├── 階層的生成モデル
├── 予測誤差の最小化
├── 精度重み付け
└── 知覚は「制御された幻覚」

↓

能動的推論
├── 行動は予測誤差最小化の一形態

└── 自由エネルギー原理との接続

↓

4E認知
├── Embodied：身体性
├── Embedded：環境への埋め込み

↓

```

## 2. 理論の力点（何を説明できるのか）

### 2.1 説明範囲

**（1）知覚の本質**

知覚がなぜ「解釈」を含むのか、錯覚がなぜ生じるのか、注意がどう機能するのかを説明する。

**（2）知覚と行動の統合**

知覚と行動がなぜ密接に結びついているのかを、単一の原理で説明する。

**（3）道具使用と技術**

人間がなぜ道具を使い、技術と融合するのかを、認知の拡張として説明する。

**（4）精神病理**

統合失調症、自閉症、不安障害などを、予測処理の異常として説明する試み。

**（5）意識と自己**

予測処理の枠組みで意識と自己感覚を説明する試み。

---

### 2.2 実践との接続点

**（1）人工知能・ロボット工学**

予測処理に基づくAIアーキテクチャの開発。身体化されたロボットの設計。

**（2）精神医学**

統合失調症を予測誤差の精度重み付けの異常として理解する試み。

**（3）ヒューマン・コンピュータ・インタラクション**

認知の拡張を考慮したインターフェース設計。

**（4）教育**

認知ツールの設計と活用。外的リソースを活かした学習環境。

補助具や技術を認知の拡張として積極的に活用する視点。

## 3. 誤解されやすい点（境界線）

### 3.1 「すべてが認知の一部」ではない

拡張された心を「すべての道具が認知の一部」と過度に拡大解釈する誤解がある。

クラークの立場：
- 特定の条件を満たすものだけが認知の一部となる
- 信頼可能性、自動的アクセスなどの条件がある
- 単に使っているだけでは不十分

### 3.2 「脳は重要でない」という誤解

拡張された心を「脳は特別ではない」という主張と誤解することがある。

クラークの立場：
- 脳は依然として認知の中核的な構成要素
- 脳には特別な種類の貢献がある
- 拡張は脳を置き換えるのではなく、脳と協働する

### 3.3 予測処理は「単なる予測」ではない

予測処理を「脳が未来を予測する」という単純な話と誤解することがある。

より正確には：
- 予測は感覚入力についての予測（何が来るか）
- 階層的で複雑な構造を持つ
- 予測誤差の処理が核心

### 3.4 「証明された理論」ではない

予測処理を確立された事実として扱う傾向がある。

現状：
- 有望な理論的枠組み
- 多くの現象を統一的に説明できる
- しかし、直接的な実証は限定的
- 反論や代替理論も存在する

## 4. 実践での留意点（三角コーン）

### 4.1 理論の抽象度を認識する

クラークの理論は高度に抽象的であり、具体的な実践への適用には翻訳が必要。

注意点：
- 直接的な教育方法や治療技法を導出するわけではない
- 「予測処理に基づく○○」という主張には慎重に
- 理論と実践の間には多くの媒介が必要

### 4.2 拡張された心の倫理的含意を考慮する

認知の拡張には倫理的な含意がある。

考慮点：
- スマートフォンを取り上げることは「記憶を奪う」ことか？
- 認知的エンハンスメントの公平性
- プライバシーと認知の境界
- 技術への依存の問題

### 4.3 精神病理への適用は慎重に

予測処理による精神病理の説明は有望だが、発展途上。

注意点：
- 直接的な治療法が導出されるわけではない
- 従来の理解を置き換えるものではなく、補完するもの
- 個々の患者への適用には慎重さが必要

### 4.4 技術楽観主義を避ける

「技術は認知を拡張するから良い」という単純な結論を避ける。

考慮点：
- 拡張されることが常に望ましいわけではない
- 技術への依存の問題
- スキルの喪失の可能性
- 誰のための拡張か

### 4.5 哲学と経験科学の関係を認識する

クラークの議論は、哲学的議論と経験科学的主張が混在している。

区別：
- 概念的・哲学的主張（認知の境界はどこか）
- 経験的主張（脳は予測処理を行っている）
- 両者は異なる種類の評価を必要とする
